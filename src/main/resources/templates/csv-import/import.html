Формат CSV файла:

Файл должен содержать заголовок и данные в следующем формате:

region_code;location_name_ru;apk_device_number;apk_cert_number;apk_cert_from;apk_cert_to;camera_ip;camera_code;camera_direction
Правила обработки данных:
  • region_code - код региона (должен существовать в системе)
  • location_name_ru - название локации на русском языке
  • apk_device_number - номер устройства АПК
  • apk_cert_number - номер сертификата АПК
  • apk_cert_from - дата выдачи сертификата (формат: yyyy-MM-dd HH:mm:ss)
  • apk_cert_to - дата окончания сертификата (формат: yyyy-MM-dd HH:mm:ss)
  • camera_ip - IP адрес камеры
  • camera_code - код камеры
  • camera_direction - направление камеры
Логика обработки:
  • Если регион с указанным кодом не существует - ошибка
  • Если локация с указанным названием не существует - создается новая
  • Если АПК с указанным номером устройства существует - обновляется, иначе создается новый
  • Если камера с указанным кодом или IP существует - обновляется, иначе создается новая
Пример CSV файла:
region_code;location_name_ru;apk_device_number;apk_cert_number;apk_cert_from;apk_cert_to;camera_ip;camera_code;camera_direction
