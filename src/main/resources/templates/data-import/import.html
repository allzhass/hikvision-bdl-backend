Инструкция по импорту:
Формат Excel файла:

Название вкладки должно соответствовать коду региона

Таблица во вкладке должна содержать заголовок и данные в следующем формате:

№,IP камеры,Код камеры,Номер устройства,Код сертификата,начало,конец,Адрес,направление
Правила обработки данных:
  • № - номер строки
  • IP камеры - IP адрес камеры
  • Код камеры - код камеры
  • Номер устройства - номер устройства АПК
  • Код сертификата - номер сертификата АПК
  • начало - дата выдачи сертификата (формат: yyyy-MM-dd HH:mm:ss)
  • конец - дата окончания сертификата (формат: yyyy-MM-dd HH:mm:ss)
  • Адрес - адрес локации
  • направление - направление камеры
Логика обработки:
  • Если регион с указанным кодом не существует - ошибка
  • Если локация с указанным названием не существует - создается новая
  • Если АПК с указанным номером устройства существует - обновляется, иначе создается новый
  • Если камера с указанным кодом или IP существует - обновляется, иначе создается новая
Пример Excel файла:
                                        №,IP камеры,Код камеры,Номер устройства,Код сертификата,начало,конец,Адрес,направление

2,192.168.1.101,CAM001,APK002,CERT002,2024-01-01 00:00:00,2025-01-01 00:00:00,Алматы центр,Юг
